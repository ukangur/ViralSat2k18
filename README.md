

ViralSAT
=================

ViralSAT is a work-in-progress crowdsourcing application, that pinpoints instagram posts
based on location and time. All the NASA Space App challenge original content is posted
under the NASA SPACE APP folder.

Install
-------
To install instagram-scraper:
```bash
$ pip install instagram-scraper
```

To update instagram-scraper:
```bash
$ pip install instagram-scraper --upgrade
```
Alternatively, you can clone the project and run the following command to install:
Make sure you cd into the *instagram-scraper-master* folder before performing the command below.
```
$ python setup.py install
```


Usage
-----

After you have installed instagram scraper, you can get to scraping. I made a simple command line converter,
